============
Installation
============


Stable release
--------------

To install NLP_emerging_technologies, run this command in your terminal:

.. code-block:: console

    $ pip install nlp_emerging_technologies

This is the preferred method to install NLP_emerging_technologies, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for NLP_emerging_technologies can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/AntoineAMC/nlp_emerging_technologies

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/AntoineAMC/nlp_emerging_technologies/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/AntoineAMC/nlp_emerging_technologies
.. _tarball: https://github.com/AntoineAMC/nlp_emerging_technologies/tarball/master
